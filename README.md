## Installation

### Using npm

1. Installation npm (global):

   ```shell
   npm i -g npm
   ```

2. Installation `shegit` package last version:

   ```shell
   npm install --save shegit
   ```

### In a browser

1. Download source code

2. Use in your html:

   ```html
   <script src="[your_lib_folder_path]/shegit"></script>
   ```
