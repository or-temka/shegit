# toColor

## Using

Convert a color of one type to another.

<LiveFunctionPreview
  name="toColor"
  func={toColor}
  args={[
    { name: 'color', default: '{"r": 242, "g": 52, "b": 11}' },
    { name: 'to', optional: true, default: '"cmyk-string"' },
  ]}
/>

### Arguments

| Argument        | Type                                     | Description                                                                                 | Possible Values                                                                                                                           | Example                                                                          |
| --------------- | ---------------------------------------- | ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| `color`         | `number[]` <br/> `object` <br/> `string` | RGB color.                                                                                  |                                                                                                                                           | `[15, 76, 213]` <br/> `{r: 255, g: 14, b: 148 }` <br/> `"cmyk(63, 69, 5.5, 34)"` |
| `to`            | `string`                                 | Return type.                                                                                | `"object"` <br/> `"rgb-object"` <br/> `"rgb-array"` <br/> `"rgb-string"` <br/> `"cmyk-object"` <br/> `"cmyk-array"` <br/> `"cmyk-string"` | `"cmyk-array"`                                                                   |
| `fromColorType` | `string`                                 | The color from which we will get the result. Can help when the source value is not defined. | `"rgb"` <br/> `"cmyk"`                                                                                                                    | `"cmyk"`                                                                         |

### Returns

`number[]` | `object` | `string` â€” Returns the color in one of the formats.

### Examples

#### Quick view

```js
toColor([93, 255, 101], 'rgb-object');
// {r: 93, g: 255, b: 101}

toColor({ c: 64, m: 21, y: 99, k: 2 }, 'rgb-array');
// [90, 197, 2]

toColor({ r: 9.8, g: 32, b: 11 }, 'rgb-string');
// rgb(9, 32, 11)

toColor('rgb(190,88,17)', 'cmyk-string');
// 'cmyk(0%, 54%, 91%, 25%)'
```

<Details headerText='To "object" or "rgb-object"' style={DETAILS_DEFAULT_STYLES}>

:::caution
Use `"rgb-object"` instead of `"object"`, as `"object"` will soon become obsolete.
:::

```js
toColor(['+53', false, true], 'object');
// {r: 53, g: 0, b: 1}

toColor([500, 177, -100], 'rgb-object');
// {r: 255, g: 177, b: 0}

toColor(() => 'rgba(23, 44, 19)', 'rgb-object');
// {r: 23, g: 44, b: 19}

toColor([0, 0, 66], 'object', fromColorType: 'cmyk');
// { r: 0, g: 0, b: 0, a: 1 }

toColor('cmyk(64%, 21%, 99%, 2%)', 'rgb-object');
// { r: 90, g: 197, b: 2, a: 1 }

toColor("some", 'rgb-object');
// { r: 0, g: 0, b: 0, a: 1 }
```

</Details>

</Details>
