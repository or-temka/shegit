# hsvObjToRgbObj

## Using

Convert HSV color `{ h: hue, s: saturation, v: value }` as an object
to a RGB color `{ r: red, g: green, b: blue, a: alpha }` as an object.

<LiveFunctionPreview
  name="hsvObjToRgbObj"
  func={hsvObjToRgbObj}
  args={[{ name: 'obj', initial: '{ "h": 220, "s": 60, "v": 89 }' }]}
/>

### Arguments

| Argument | Type                       | Description                   | Example                     |
| -------- | -------------------------- | ----------------------------- | --------------------------- |
| `obj`    | `object` <br /> `function` | HSV color object or function. | `{ h: 330, s: 100, v: 36 }` |

### Returns

`object` â€” Returns RGB object in format `{ r: red, g: green, b: blue, a: alpha }`.

### Examples

```js
hsvObjToRgbObj({ h: 220, s: 50, v: 80 });
// { r: 102, g: 136, b: 204, a: 1 }

hsvObjToRgbObj({ h: '262.3', s: '+99.9', v: '2.6' });
// { r: 2, g: 0, b: 5, a: 1 }

hsvObjToRgbObj(() => ({ h: () => -100, s: () => 500, v: 55.6 }));
// { r: 140, g: 0, b: 0, a: 1 }

hsvObjToRgbObj('John');
// { r: 0, g: 0, b: 0, a: 1 }
```
