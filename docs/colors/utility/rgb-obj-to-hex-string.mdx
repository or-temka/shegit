# rgbObjToHexString

## Using

Convert RGB color `{r: red, g: green, b: blue}` as an object
to a HEX color "#`red as 16*` `green as 16*` `blue as 16*`" as an string (`16*` - Hexagonal Calculus).

<LiveFunctionPreview
  name="rgbObjToHexString"
  func={rgbObjToHexString}
  args={[{ name: 'obj', initial: '{"r": 125, "g": 14, "b": 96}' }]}
/>

### Arguments

| Argument | Type                       | Description                   | Example                    |
| -------- | -------------------------- | ----------------------------- | -------------------------- |
| `obj`    | `object` <br /> `function` | RGB color object or function. | `{r: 209, g: 205, b: 152}` |

### Returns

`object` â€” Returns HEX string in format "#`red as 16*` `green as 16*` `blue as 16*`" (`16*` - Hexagonal Calculus).

### Examples

```js
rgbObjToHexString({ r: 64, g: 21, b: 99 });
// '#401563'

rgbObjToHexString({ r: 34.3, g: '+59.8', b: () => '-96.5' });
// '#223B00'

rgbObjToHexString(() => ({ r: () => '45.7', g: () => '+291.5', b: () => 0 }));
// '#2DFF00'

rgbObjToHexString('some');
// '#000000'
```
