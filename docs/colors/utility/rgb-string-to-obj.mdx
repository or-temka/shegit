# rgbStringToObj

## Using

Convert RGB color `"rgb(red, greed, blue)"` as an string to a format object `{r: red, g: green, b: blue, a: alpha}`

<LiveFunctionPreview
  name="rgbStringToObj"
  func={rgbStringToObj}
  args={[{ name: 'string', initial: '"rgb(85, 15, 15)"' }]}
/>

### Arguments

| Argument | Type                       | Description       | Example               |
| -------- | -------------------------- | ----------------- | --------------------- |
| `string` | `string` <br /> `function` | Rgb color string. | `"rgb(210, 193, 54)"` |

### Returns

`object` â€” Returns RGB object in format `{r: red, g: green, b: blue, a: alpha}`.

### Examples

```js
rgbStringToObj('rgb(15, 24, 99)');
// {r: 15, g: 24, b: 99, a: 1}

rgbStringToObj(() => 'rgb(-20, 1000, 7.3, 0.5)');
// {r: 0, g: 255, b: 7, a: 0.5}

rgbStringToObj('rgba(202.9, -2, 7, -100)');
// {r: 202, g: 0, b: 7, a: 0}

rgbStringToObj('87, 135, 245, 2');
// {r: 87, g: 135, b: 245, a: 1}
```
