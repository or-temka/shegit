# capitalizeFirstLetter

## Using

Function takes a string and returns a new string with the first letter capitalized.

<LiveFunctionPreview
  name="capitalizeFirstLetter"
  func={capitalizeFirstLetter}
  args={[{ name: 'str', initial: '"any string"' }]}
/>

### Arguments

| Argument | Type     | Description | Example                         |
| -------- | -------- | ----------- | ------------------------------- |
| `str`    | `string` | String      | `"some value"` <br /> `"hello"` |

### Returns

`string` | `undefined` â€” String with the first letter capitalized or `undefined` if `str` is not string.

### Examples

```js
capitalizeFirstLetter('some value for function');
// "Some value for function"

capitalizeFirstLetter('hello');
// "Hello"

capitalizeFirstLetter('Any');
// "Any"

capitalizeFirstLetter('2ok');
// "2ok"
```
