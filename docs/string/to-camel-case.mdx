| Argument | Type     | Description | Example                            |
| -------- | -------- | ----------- | ---------------------------------- |
| `str`    | `string` | String      | `"iAmString"` <br /> `"Text_TEXT"` |

### Returns

`string` | `undefined` â€” String in the format `camelCase` or `undefined` if `str` was not a string

### Examples

```js
toCamelCase('some value');
// "someValue"

toCamelCase('ANY_STRING');
// "anyString"

toCamelCase('iLikeShegit');
// "iLikeShegit"

toCamelCase();
// undefined

toCamelCase(55);
// undefined
```
