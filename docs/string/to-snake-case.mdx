# toSnakeCase

## Using

Сonverts a string into `snake_case` format.

<LiveFunctionPreview
  name="toSnakeCase"
  func={toSnakeCase}
  args={[{ name: 'str', initial: '"any string"' }]}
/>

### Arguments

| Argument | Type     | Description | Example                            |
| -------- | -------- | ----------- | ---------------------------------- |
| `str`    | `string` | String      | `"iAmString"` <br /> `"Text_TEXT"` |

### Returns

`string` | `undefined` — String in the format `snake_case` or `undefined` if `str` was not a string

### Examples

```js
toSnakeCase('some value');
// "some_value"

toSnakeCase('ANY_STRING');
// "any_string"

toSnakeCase('iLikeShegit');
// "i_like_shegit"

toSnakeCase();
// undefined

toSnakeCase(55);
// undefined
```
