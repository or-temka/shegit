### Arguments

| Argument | Type     | Description | Example                            |
| -------- | -------- | ----------- | ---------------------------------- |
| `str`    | `string` | String      | `"iAmString"` <br /> `"Text_TEXT"` |

### Returns

`string` | `undefined` â€” String in the format `snake_case` or `undefined` if `str` was not a string

### Examples

```js
toSnakeCase('some value');
// "some_value"

toSnakeCase('ANY_STRING');
// "any_string"

toSnakeCase('iLikeShegit');
// "i_like_shegit"

toSnakeCase();
// undefined

toSnakeCase(55);
// undefined
```
