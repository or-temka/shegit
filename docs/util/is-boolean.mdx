# isBoolean

## Using

This function determines whether the given value is a **primitive boolean**.

- Returns `true` only if the value is the primitive `true` or `false`.
- Returns `false` for any non-boolean type or boolean object (e.g., `new Boolean(true)`).

### Returns

`boolean` â€” `true` if value is a boolean primitive, `false` otherwise

### Examples

```js
isBoolean(true);
// true

isBoolean(false);
// true

isBoolean(new Boolean(true));
// false

isBoolean('true');
// false

isBoolean(Boolean(0));
// true

isBoolean(1 === 1);
// true

isBoolean(() => true);
// false

isBoolean(null);
// false

isBoolean({});
// false

isBoolean([true]);
// false
```
