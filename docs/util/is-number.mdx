# isNumber

## Using

This function determines whether the given value is a valid, finite primitive number.

- Returns `true` if the value is a finite number (not `NaN`, `Infinity`, or `-Infinity`).
- Returns `false` for non-number types or special numeric values.

### Arguments

| Argument | Type      | Description     | Example                                |

| `value`  | `unknown` | Can be any type | `"text"` <br /> `2` <br /> `undefined` |

### Returns

`boolean` â€” `true` if value is a finite number, `false` otherwise

### Examples

```js
isNumber(42);
// true

isNumber(0);
// true

isNumber(-3.14);
// true

isNumber('42');
// false

isNumber(NaN);
// false

isNumber(Infinity);
// false

isNumber(new Number(5));
// false

isNumber(() => {});
// false

isNumber(undefined);
// false
```
