# hsvObjToArray

## Using

Convert HSV color `{h: hue, s: saturation, v: value}` as an object to a format array `[hue, saturation, value]`

<LiveFunctionPreview
  name="hsvObjToArray"
  func={hsvObjToArray}
  args={[{ name: 'obj', default: '{"h": 220, "s": 67, "v": 96}' }]}
/>

### Arguments

| Argument | Type                       | Description                   | Example                  |
| -------- | -------------------------- | ----------------------------- | ------------------------ |
| `obj`    | `object` <br /> `function` | HSV color object or function. | `{h: 300, s: 10, v: 45}` |

### Returns

`array` â€” Returns HSV array in format `[hue, saturation, value]`.

### Examples

```js
hsvObjToArray({ h: -22, s: 99, v: '39' });
// [0, 99, 39]

hsvObjToArray({ h: () => 2, s: () => 55, v: () => 1005 });
// [2, 55, 100]

hsvObjToArray('some');
// [0, 0, 0]
```
