# rgbObjToCmykObj

## Using

Convert RGB color `{r: red, g: green, b: blue}` as an object
to a CMYK color `{c: cyan, m: magenta, y: yellow, k: key }` as an object.

<LiveFunctionPreview
  name="rgbObjToCmykObj"
  func={rgbObjToCmykObj}
  args={[{ name: 'obj', default: '{"r": 125, "g": 14, "b": 96}' }]}
/>

### Arguments

| Argument | Type                       | Description                   | Example                    |
| -------- | -------------------------- | ----------------------------- | -------------------------- |
| `obj`    | `object` <br /> `function` | Rgb color object or function. | `{r: 209, g: 205, b: 152}` |

### Returns

`object` â€” Returns CMYK object in format `{c: cyan, m: magenta, y: yellow, k: key }`.

### Examples

```js
rgbObjToCmykObj({ r: 64, g: 21, b: 99 });
// { c: 35, m: 79, y: 0, k: 61 }

rgbObjToCmykObj({ r: '34.3', g: '+59.8', b: -96.5 });
// { c: 42, m: 0, y: 100, k: 77 }

rgbObjToCmykObj(() => ({ r: () => false, g: () => true, b: () => 255 }));
// { c: 100, m: 100, y: 0, k: 0 }

rgbObjToCmykObj('some');
// { c: 0, m: 0, y: 0, k: 100 }
```
