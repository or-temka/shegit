# hsvArrayToObj

## Using

Convert HSV color `[hue, saturation, value]` as an array to a format object `{h: hue, s: saturation, v: value}`

<LiveFunctionPreview
  name="hsvArrayToObj"
  func={hsvArrayToObj}
  args={[{ name: 'array', initial: '[220, 67, 96]' }]}
/>

### Arguments

| Argument | Type                      | Description                  | Example         |
| -------- | ------------------------- | ---------------------------- | --------------- |
| `array`  | `array` <br /> `function` | HSV color array or function. | `[300, 10, 45]` |

### Returns

`object` â€” Returns HSV object in format `{h: hue, s: saturation, v: value}`.

### Examples

```js
hsvArrayToObj([-22, 99, '39']);
// { h: 0, s: 99, v: 39 }

hsvArrayToObj(() => [() => 2, 55.5, () => '1005']);
// { h: 2, s: 55, v: 100 }

hsvArrayToObj('some');
// { h: 0, s: 0, v: 0 }
```
