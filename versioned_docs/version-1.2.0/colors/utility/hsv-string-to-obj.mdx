# hsvStringToObj

## Using

Convert HSV color "hsv(`hue`, `saturation` %, `value` %)" as an string to a format object `{ h: hue, s: saturation, v: value }`

<LiveFunctionPreview
  name="hsvStringToObj"
  func={hsvStringToObj}
  args={[{ name: 'string', initial: '"hsv(180, 90%, 40%)"' }]}
/>

### Arguments

| Argument | Type                       | Description                   | Example          |
| -------- | -------------------------- | ----------------------------- | ---------------- |
| `string` | `string` <br /> `function` | HSV color string or function. | `hsv(83, 14, 9)` |

### Returns

`object` â€” Returns HSV object in format `{ h: hue, s: saturation, v: value }`.

### Examples

```js
hsvStringToObj('hsv(-200, 99.999%, +39.2)');
// { h: 0, s: 99, v: 39 }

hsvStringToObj(() => '2, 55, 999');
// { h: 2, s: 55, v: 100 }

hsvStringToObj('some');
// { h: 0, s: 0, v: 0 }
```
