# rgbObjToString

## Using

Convert RGB color `{r: red, g: green, b: blue}` as an object to a format string `"rgb(red, green, blue)"`

<LiveFunctionPreview
  name="rgbObjToString"
  func={rgbObjToString}
  args={[{ name: 'obj', initial: '{"r": 199, "g": 85, "b": 4}' }]}
/>

### Arguments

| Argument | Type                       | Description       | Example                    |
| -------- | -------------------------- | ----------------- | -------------------------- |
| `obj`    | `object` <br /> `function` | Rgb color object. | `{r: 209, g: 205, b: 152}` |

### Returns

`string` â€” Returns RGB string in format `"rgb(red, green, blue)"` (`red`, `green`, `blue` = number).

### Examples

```js
rgbObjToString({ r: 15, g: 76, b: 213 });
// rgb(15, 76, 213)

rgbObjToString({ r: '+99', g: 300, b: -5 });
// rgb(99, 255, 0)

rgbObjToString({ r: () => true, g: false, b: 245.9 });
// rgb(1, 0, 245)

rgbObjToString(() => ({ r: 3.2, g: () => 55, b: '184' }));
// rgb(3, 55, 184)
```
