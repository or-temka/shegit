# cmykArrayToObj

## Using

Convert CMYK color `[cyan, magenta, yellow, key]` as an array to a format object `{c: cyan, m: magenta, y: yellow, k: key}`

<LiveFunctionPreview
  name="cmykArrayToObj"
  func={cmykArrayToObj}
  args={[{ name: 'array', initial: '[85, 91, 19, 56]' }]}
/>

### Arguments

| Argument | Type                         | Description       | Example         |
| -------- | ---------------------------- | ----------------- | --------------- |
| `array`  | `number[]` <br /> `function` | CMYK color array. | `[15, 76, 213]` |

### Returns

`object` â€” Returns CMYK object in format `{c: cyan, m: magenta, y: yellow, k: key}`.

### Examples

```js
cmykArrayToObj([15, 76, 75, 50]);
// {c: 15, m: 76, y: 75, k: 50}

cmykArrayToObj(() => [500, () => 13, -1000, '-5']);
// {c: 100, m: 13, y: 0, k: 0}

cmykArrayToObj(['99', '+45', false, true]);
// {c: 99, m: 45, y: 0, k: 1}

cmykArrayToObj([99, 245, 0, 5.9]);
// {c: 99, m: 100, y: 0, k: 5}
```
