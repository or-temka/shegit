# cmykObjToArray

## Using

Convert CMYK color `{c: cyan, m: magenta, y: yellow, k: key }` as an object to a format array `[cyan, magenta, yellow, key]`

<LiveFunctionPreview
  name="cmykObjToArray"
  func={cmykObjToArray}
  args={[{ name: 'obj', initial: '{"c": 255, "m": 14, "y": 96, "k": 59}' }]}
/>

### Arguments

| Argument | Type                       | Description        | Example                        |
| -------- | -------------------------- | ------------------ | ------------------------------ |
| `obj`    | `object` <br /> `function` | CMYK color object. | `{c: 10, m: 24, y: 57, k: 96}` |

### Returns

`array` â€” Returns CMYK array in format `[cyan, magenta, yellow, key]`.

### Examples

```js
cmykObjToArray({ c: 15, m: 76, y: 14, k: 50 });
// [15, 76, 14, 50]

cmykObjToArray({ c: '+99', m: 300, y: -5, k: 99 });
// [99, 100, 0, 99]

cmykObjToArray({ c: () => true, m: false, y: 45.9, k: NaN });
// [1, 0, 45, 0]

cmykObjToArray(() => ({ c: 3.2, m: () => 55, y: '84', k: '-7' }));
// [3, 55, 84, 0]
```
