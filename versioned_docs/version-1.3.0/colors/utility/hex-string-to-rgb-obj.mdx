# hexStringToRgbObj

## Using

Convert HEX color "#`red as 16*` `green as 16*` `blue as 16*`" as an string (`16*` - Hexagonal Calculus)
to a RGB color `{r: red, g: green, b: blue}` as an object.

  name="hexStringToRgbObj"
  func={hexStringToRgbObj}

/>

### Arguments

| Argument | Type                       | Description                   | Example   |
| -------- | -------------------------- | ----------------------------- | --------- |
| `str`    | `string` <br /> `function` | HEX color string or function. | `#f80032` |

### Returns

`object` â€” Returns RGB color in format `{r: red, g: green, b: blue}` as an string.

### Examples

```js
hexStringToRgbObj('#1200E3');
// { r: 18, g: 0, b: 227 }

hexStringToRgbObj(() => '#401563');
// { r: 64, g: 21, b: 99 }

hexStringToRgbObj(false);
// '#000000'
```
