# hsvObjToString

## Using

Convert HSV color `{h: hue, s: saturation, v: value}` as an object to a format array "hsv(`hue`, `saturation`%, `value`%)".

<LiveFunctionPreview
  name="hsvObjToString"
  func={hsvObjToString}
  args={[{ name: 'obj', initial: '{"h": 220, "s": 67, "v": 96}' }]}
/>

### Arguments

| Argument | Type                       | Description                   | Example                  |
| -------- | -------------------------- | ----------------------------- | ------------------------ |
| `obj`    | `object` <br /> `function` | HSV color object or function. | `{h: 300, s: 10, v: 45}` |

### Returns

`string` â€” Returns HSV string in format "hsv(`hue`, `saturation`%, `value`%)".

### Examples

```js
hsvObjToString({ h: -22, s: 99, v: '39' });
// "hsv(0, 99%, 39%)"

hsvObjToString({ h: () => 2, s: () => 55, v: () => 1005 });
// "hsv(2, 55%, 100%)"

hsvObjToString('some');
// "hsv(0, 0%, 0%)"
```
