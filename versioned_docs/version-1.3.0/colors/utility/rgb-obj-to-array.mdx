# rgbObjToArray

## Using

Convert RGB color `{r: red, g: green, b: blue}` as an object to a format array `[red, green, blue]`

<LiveFunctionPreview
  name="rgbObjToArray"
  func={rgbObjToArray}
  args={[{ name: 'obj', initial: '{"r": 255, "g": 14, "b": 96}' }]}
/>

### Arguments

| Argument | Type                       | Description       | Example                    |
| -------- | -------------------------- | ----------------- | -------------------------- |
| `obj`    | `object` <br /> `function` | Rgb color object. | `{r: 209, g: 205, b: 152}` |

### Returns

`array` â€” Returns RGB array in format `[red, green, blue]`.

### Examples

```js
rgbObjToArray({ r: 15, g: 76, b: 213 });
// [15, 76, 213]

rgbObjToArray({ r: '+99', g: 300, b: -5 });
// [99, 255, 0]

rgbObjToArray({ r: () => true, g: false, b: 245.9 });
// [1, 0, 245]

rgbObjToArray(() => ({ r: 3.2, g: () => 55, b: '184' }));
// [3, 55, 184]
```
