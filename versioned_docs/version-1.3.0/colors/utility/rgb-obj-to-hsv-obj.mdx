# rgbObjToHsvObj

## Using

Convert RGB color `{r: red, g: green, b: blue}` as an object
to a HSV color `{ h: hue, s: saturation, v: value }` as an object.

<LiveFunctionPreview
  name="rgbObjToHsvObj"
  func={rgbObjToHsvObj}
  args={[{ name: 'obj', initial: '{"r": 125, "g": 14, "b": 96}' }]}
/>

### Arguments

| Argument | Type                       | Description                   | Example                    |
| -------- | -------------------------- | ----------------------------- | -------------------------- |
| `obj`    | `object` <br /> `function` | RGB color object or function. | `{r: 209, g: 205, b: 152}` |

### Returns

`object` â€” Returns HSV object in format `{ h: hue, s: saturation, v: value }`.

### Examples

```js
rgbObjToHsvObj({ r: 64, g: 21, b: 99 });
// { h: 273, s: 79, v: 39 }

rgbObjToHsvObj({ r: '34.3', g: '+59.8', b: -96.5 });
// { h: 85, s: 100, v: 23 }

rgbObjToHsvObj(() => ({ r: () => false, g: () => true, b: () => 255 }));
// { h: 240, s: 100, v: 100 }

rgbObjToHsvObj('some');
// { h: 0, s: 0, v: 0 }
```
