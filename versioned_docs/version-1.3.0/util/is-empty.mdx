# isEmpty

## Using

This function to determine if a given value is empty.

- Return `true` if the value is considered empty.
- Return `false` otherwise.

<LiveFunctionPreview
  name="isEmpty"
  func={isEmpty}
  args={[{ name: 'value', initial: '[ "any string", 12 ]' }]}
/>

### Arguments

| Argument | Type  | Description     | Example                                 |
| -------- | ----- | --------------- | --------------------------------------- |
| `value`  | `any` | Can be any type | `"text"` <br /> `{}` <br /> `new Map()` |

### Returns

`boolean` â€” value is empty `true | false`

### Examples

```js
isEmpty(undefined);
// true

isEmpty(34);
// false

isEmpty([]);
// true

isEmpty([1, 2]);
// false

isEmpty({});
// true

isEmpty({ name: 'Alina' });
// false

isEmpty(null);
// true

isEmpty('');
// true

isEmpty('some');
// false
```
